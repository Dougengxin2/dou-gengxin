Vue项目基础流程

Author: Dou Gengxin Date: February 4, 2019  23:16:31 Category: Vue

项目流程

一、安装node.js

查看安装好的node.js node -v

二、安装webpack.js

查看安装好的webpack webpack -v

三、安装npm（npm i）

查看安装好的npm npm -v

四、get部署项目（需要下载安装git）

复制项目地址，在项目文件夹下右键打开Git Bash Here 进入命令框，输入 git clone （复制的地址），将git上的文件克隆到本地，会生成一个.git的隐藏文件，在这个带有隐藏文件的文件夹下创建项目文件夹，方可提交到此项目的git仓库

五、下载vue全家桶

npm i vue vue-cli vue-router vuex -S （vue-cli脚手架 vue-router是路由 vuex是存储数据）

六、创建脚手架

运行 vue init webpack

七、完成以上补足项目已经创建完毕可在项目文件夹下使用 npm run dev 运行项目

八、若需要element-ui则下载

npm i element-ui -S
